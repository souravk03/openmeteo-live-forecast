Hybrid LSTM-GRU (UV)
Combines LSTM's long-term memory with GRU's efficiency on a single variable. Trained on 5+ years of ERA5 history.
🔵 Univariate
⚡
GRU (UV)
Gated Recurrent Unit — lighter than LSTM, captures medium-range temporal dependencies on single-variable sequences.
🔵 Univariate
🕸️
ANN (UV)
Feedforward neural network on lagged single-variable features. Fast non-sequential baseline in the UV ensemble.
🔵 Univariate
🔗
Hybrid LSTM-GRU (MV)
Takes all variables (TMP2m, RH2m, PRmsl) jointly — learns cross-variable meteorological dynamics.
🟣 Multivariate
🎯
GRU (MV)
Multivariate GRU over 133 engineered columns — strong at catching multi-variable co-movement patterns.
🟣 Multivariate
🧩
ANN (MV)
Feedforward ANN on all 133 features including lags, rolling stats, and cross-feature interactions.
🟣 Multivariate
